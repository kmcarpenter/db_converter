Command Line Options

Uninstall

This option will load the Windows Installer. This is the same as choosing the Uninstall option from the menu.

Help

This option will show the above screenshot. This will also be shown if you enter an incorrect option.

Convert

This option has 4 Sub-Options:
[ConversionType] - Choose MySQL, Oracle, or PostgreSQL. This option is required.
[AccessDB] - The path and name of a MS Access Database to convert. This option is required.
[OutputFile] - The path and name of an output file. This is optional. It will be assumed if you do not include it.
[IncludeData] - Whether or not to include data. Yes or No. This is set to Yes by default.
