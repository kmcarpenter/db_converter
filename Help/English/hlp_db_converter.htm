This form contains all the options you can use to change DB Converter's functionality.

General Options

Check Boxes

Assume Output Filename

Check to have DB Converter assume output filename and directory.

Remember Options

Check to retain settings when you restart wizard.

Convert Return

If return is pressed in a Textbox, convert it to TAB (go to next box).

Highlight Text

When a textbox gets focus, highlight the text in it.

Show Tips

Show "Tip of the Day" on Startup.

Donations

DB Converter is GPL software. This means it is free, and nobody can charge you (other than for distribution) for it. All that we ask is for a simple donation.

Don't Warn Linked Tables

Linked tables don't tell DB Converter how many records they contain. This means they cannot be converted. If this option is turned on, you will not receive a warning about this issue.

Convert All Tables

If this option is checked, you will not be asked what tables you want to convert. They will all be converted.

Check for Updates

Choosing this option means that DB Converter will check to see if a new version is available every time you start the program.

INNODB

By turning this option on, you will tailor your MySQL output to use their INNODB type of table. Otherwise the default MyISAM is used.
