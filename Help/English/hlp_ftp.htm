This page contains help for both the FTP and MySQL forms. It will explain all the options you can use to have DB Converter connect to remote hosts.

FTP Form
Host Remote address of ftp server to connect to.
Port Remote port of ftp server (21 is FTP default).
User Remote user name to connect with.
Password Remote password for above user name.

MySQL Form

Host Remote address of mysql server to connect to.
Port Remote port of ftp server (3306 is MySQL default).
Database Remote database to connect to.
User Remote user name to connect with.
Password Remote password to connect with.
