Welcome to DB Converter!

This tool will allow easy conversion from MS Access Databases to MySQL or Oracle Databases. We've always found it a pain having to write the SQL to create one of these databases when you already have another database we want to model it after.

Now you don't have to write the SQL! DB Converter will do it all for you. It dramatically saves time allowing you to finish projects faster!
